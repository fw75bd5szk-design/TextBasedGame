# Gates & Hunters

## Project Overview

Gates & Hunters is a Python text-based adventure game where the player explores different rooms, collects items, unlocks paths, and prepares to face the final villain, the Monarch of Destruction. The goal of the game is to find the Ancient Relic before entering the Boss Chamber. If the player enters the Boss Chamber too early, they may lose unless they have the Resurrection Stone.

## How to Run the Program

1. Make sure Python is installed on your computer.
2. Download or clone this repository.
3. Open the project folder in your IDE or terminal.
4. Run the Python file:

```bash
